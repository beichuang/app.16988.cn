# php-qrcode 在线生成二维码

基于phpqrcode+js实现的在线生成二维码工具(canvas版)，个人已使用许久。

目前支持的功能：

  随意更换颜色、可自定义版权文案、可一键下载二维码，支持页面刷新等。
